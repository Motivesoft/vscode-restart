# vscode-restart

Restart or Quit Code from the Command Palette or with a Keyboard Shortcut.

## Features

Adds commands to the Command Palette that allow you to quit or restart Code, either immediately or with a yes/no prompt.

From within the Command Palette, begin type either `restart` or `quit` and you will see the available commands in the list. 

These commands can be executed directly from the Command Palette or bound to a Keyboard Shortcut.
